Analizador Léxico-sintáctico

Información del proyecto

  • Lenguaje: Java

En este trabajo se realizó un analizador Léxico-sintáctico el cual se encarga de leer un archivo que contenga código en lenguaje Python y realizar la primera fase del análisis de código siendo la parte léxica donde se extraen los tokens de ese mismo código fuente, los clasifica en identificadores, palabras reservadas, operadores, etc. y muestra el lexema correspondiente a cada token. Después se realiza el análisis sintáctico donde se verifica que se cumplan con las reglas gramaticales especificadas.
