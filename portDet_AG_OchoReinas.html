Algoritmo genético de problema de las ocho reinas

Información del proyecto

  • Lenguaje: Java

Una tarea que realice fue la de resolver el problema de las ocho reinas haciendo la implementación de un algoritmo genético, para ello use como individuo un array en donde los índices y el valor que se almacenaba en ese lugar se tomaba como el par ordenado que indicaba la ubicación de la reina dentro del tablero de ajedrez. Este programa permite elegir el número de reinas que se desee siempre y cuando sea mayor o igual a 4, así también se puede especificar el tamaño de la población y cuantas generaciones habrá. Además, al correr el algoritmo este se detiene cuando encuentra la solución más optima o en caso contrario realiza todas las generaciones.
