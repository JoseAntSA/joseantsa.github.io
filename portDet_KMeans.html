Clasificador K-Means

Información del proyecto

  • Lenguaje: Java

Trabajo donde se implementó el algoritmo de clasificación no supervisada K-means que agrupa objetos en k grupos basándose en sus características. El agrupamiento se realiza minimizando la suma de distancias entre cada objeto y el centroide de su grupo o clúster, en este caso se usó la distancia euclidiana para sacar la distancia entre los puntos.
